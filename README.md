# Spam/Ham Message Classifier

A Python project to classify messages as **spam** or **ham (legitimate)** using **Machine Learning**.

## Tech Stack
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Logistic Regression

## Dataset
Sample dataset is included in `data/messages.csv`.  
Columns:
- `label` → 'spam' or 'ham'
- `message` → the text message
